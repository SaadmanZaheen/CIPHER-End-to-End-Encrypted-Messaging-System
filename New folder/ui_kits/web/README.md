# SecureConnect Web UI Kit

A high-fidelity, click-through prototype of the SecureConnect web application.

## Structure

- `index.html` — Interactive prototype (Login → Feed → Messages → Network → Profile)
- `NavBar.jsx` — Glassmorphism sticky navbar with badge counts
- `AuthScreens.jsx` — Login, Register, OTP Setup, OTP Verify screens
- `FeedScreen.jsx` — Feed with compose card, post cards, locked posts
- `MessagesScreen.jsx` — Inbox list + conversation view
- `NetworkScreen.jsx` — Network list + Discover grid + pending requests
- `ProfileScreen.jsx` — Profile details, key info, edit form

## Design width: 1200px (full viewport, max-width 900px container)
## Font: Inter (Google Fonts)
## Framework: Bootstrap 5.3 dark + custom CSS from base.html

## Screens covered
1. Login
2. Register
3. OTP Setup (2FA)
4. OTP Verify
5. Feed (with compose + post cards + locked post)
6. Discover People
7. Network / Friends (with pending requests)
8. Messages Inbox
9. Conversation / Chat
10. Profile
11. Key Rotation

## Usage
Open `index.html` in a browser. Click through the prototype — all navigation is wired.
